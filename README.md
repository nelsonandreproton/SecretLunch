# Team Member Random Selector

A small client-side web application that fairly selects one random member per team.

## Features
- Create teams
- Add members to teams
- Fair random selection (no repeats until all members are selected)
- Fully static and deployable on GitHub Pages

## Usage
Open `index.html` locally or deploy to GitHub Pages.

## Deployment
Enable GitHub Pages on the `main` branch, root folder.
